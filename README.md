# ARTA : 사용자 맞춤형 온라인 전시회 플랫폼

ARTA는 '예술'의 영문인 'art'와, '~로 부터'라는 뜻을 가진 영문 단어 'a'를 합하여 만든 합성어로 '예술로부터'라는 뜻을 가지고 있습니다.
